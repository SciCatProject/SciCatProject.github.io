A high performance petabyte metadata catalog designed for neutron and photon facilities

Scicat is a metadata catalogue. It allows users access to information about experimental results measured at your facility. Scientific datasets are linked to proposals, samples, publications, policies, attachments and many more models. SciCat helps to keep track of data provenance (i.e. the steps leading to the final results). Scicat allows users to find data based on the metadata (both your own data and other peoplesâ€™ public data). SciCat helps to automate scientific analysis workflows.

SciCat is open source, web based and cross platform
